Extremely Micro Trailer

In this project, we generate extremely micro trailers as video covers in vision perspective to replace static images covers, make videos more attractive, and improve CTR of newly-uploaded videos. Currently, each micro trailer consists of 3 seperated shots from a video. Trailers are constructed by three extracted shots in a rearranged order. One notable thing is that these trailers are generated from newly uploaded videos, which means that there are no user-interaction or statistic data for generation. More details would be published later.

For furthur exploration, we would try to utilize title/tags (Semantic Feature) set by uploaders/youtubers to build a novel pipeline with better performance.
